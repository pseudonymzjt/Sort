欢迎来到我的排序算法题解仓库！

这个项目旨在记录和整理我在学习数据结构与算法过程中，遇到的各种排序算法问题的实现和笔记。希望它不仅能作为我个人的学习档案，也能为其他正在学习排序算法的朋友们提供一些参考。

---

## ✨ 特点

- **多种实现**: 涵盖了主流的排序算法，如冒泡、选择、插入、归并、快速排序等。
- **清晰注释**: 每份代码都附有详细的注释，解释了算法的思路和关键步骤。
- **复杂度分析**: 包含了每种算法的时间和空间复杂度分析。
- **持续更新**: 会不断添加新的排序算法或相关问题的题解。

---

## 🗂️ 算法一览

下表总结了本仓库中可能包含的排序算法及其关键特性。

| 算法 (Algorithm) | 平均时间复杂度 | 最坏时间复杂度 | 空间复杂度 | 稳定性 |
| :--------------- | :------------- | :------------- | :--------- | :----- |
| 🫧 **冒泡排序** (Bubble Sort) | O(n²) | O(n²) | O(1) | 稳定 |
| 🎯 **选择排序** (Selection Sort)| O(n²) | O(n²) | O(1) | 不稳定 |
| 🃏 **插入排序** (Insertion Sort)| O(n²) | O(n²) | O(1) | 稳定 |
| 🐚 **希尔排序** (Shell Sort) | O(n log n) | O(n²) | O(1) | 不稳定 |
| 🧩 **归并排序** (Merge Sort) | O(n log n) | O(n log n) | O(n) | 稳定 |
| ⚡️ **快速排序** (Quick Sort) | O(n log n) | O(n²) | O(log n) | 不稳定 |
| 🌳 **堆排序** (Heap Sort) | O(n log n) | O(n log n) | O(1) | 不稳定 |
| 🪣 **桶排序** (Bucket Sort) | O(n+k) | O(n²) | O(n+k) | 稳定 |

*(注意: 请根据你实际实现的算法来更新此表格)*

## 🚀 快速开始

1.  **克隆仓库**
    ```bash
    git clone https://github.com/pseudonymzjt/Sort.git
    ```

2.  **进入指定算法目录**
    ```bash
    cd your-repo-name/quick_sort
    ```

3.  **运行代码**
    *   对于 Python:
        ```bash
        python solution.py
        ```
    *   对于 C++:
        ```bash
        g++ solution.cpp -o solution && ./solution
        ```
